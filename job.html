Senior Frontend Developer

Remote Full-time $110K-$140K
×

Overview

We are looking for an experienced Senior Frontend Developer to join our team. You will be responsible for building the client-side of our web applications with a focus on responsive design, performance, and user experience.

Responsibilities

  • Develop new user-facing features using React.js
  • Build reusable components and front-end libraries
  • Translate designs and wireframes into high-quality code
  • Optimize components for maximum performance across devices
  • Collaborate with UX/UI designers to implement design systems
  • Participate in code reviews and mentor junior developers

Requirements

  • 5+ years of experience in frontend development
  • Strong proficiency in JavaScript, including DOM manipulation
  • Thorough understanding of React.js and its core principles
  • Experience with state management (Redux, MobX, or Context API)
  • Familiarity with modern frontend build pipelines and tools
  • Experience with responsive design and cross-browser compatibility
  • Knowledge of RESTful APIs and modern authorization mechanisms

Cloud Security Engineer

Austin, TX Full-time $130K-$160K
×

Overview

We're seeking a Cloud Security Engineer to design and implement security controls for our cloud infrastructure. You'll play a critical role in protecting our systems and data across AWS and Azure environments.

Responsibilities

  • Design and implement security architectures for cloud environments
  • Develop and maintain cloud security policies and standards
  • Implement and manage security tools (WAF, IDS/IPS, SIEM)
  • Conduct security assessments and penetration testing
  • Monitor cloud environments for security incidents
  • Implement identity and access management solutions
  • Collaborate with DevOps teams on secure CI/CD pipelines

Requirements

  • 4+ years of experience in cloud security
  • Expertise in AWS and Azure security services
  • Experience with infrastructure as code (Terraform, CloudFormation)
  • Knowledge of container security (Docker, Kubernetes)
  • Understanding of network security and encryption protocols
  • Relevant certifications (CISSP, CCSP, AWS Security Specialty)
  • Experience with security compliance frameworks (ISO 27001, SOC 2)
